# electron-vue-project

## 项目介绍
```
用于快速创建基于VueCli3的Electron桌面程序。
```

### 项目依赖安装
```
npm install
```

### 运行开发环境
```
npm run e:d
```

### 运行打包
```
npm run e:b
```

### 安装软件
```
npm run postinstall
```

### 卸载软件
```
npm run postuninstall
```
